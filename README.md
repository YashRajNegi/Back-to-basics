# Back-to-basics

A web development project focusing on backend fundamentals.

## Project Structure

```
backend/
  ├── controllers/
  │   └── userController.js
  ├── routes/
  │   └── userRoutes.js
  ├── app.js
  ├── index.js
  └── package.json
```

## Setup

1. Navigate to the backend directory:
```bash
cd backend
```

2. Install dependencies:
```bash
npm install
```

3. Start the server:
```bash
node index.js
```

## Technologies Used

- Node.js
- Express.js
